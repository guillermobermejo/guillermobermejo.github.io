Project 2 of 3 (Software Engineering)

Political Gerrymandering State Analyzer

Full-Stack Web App

My Role: Front-End Developer


More Details
Brief Description:

Team based web application developed at Stony Brook University. A gerrymandering analyzer for testing 'fairness' of proposed and active state districting plans through a variety of tests including efficiency gap. The plan in question is compared to an 'average' plan generated from the average of 10,000 plans generated by the 'Seawulf.

Technologies used:

JavaScript, HTML, CSS, Bootstrap, GEOjson, Frappe framework, Plotly framework, Leaflet Javascript API, Fetch API, LucidChart, Github, Canva, Python, Java, Spring Boot, NoSQL, Stony Brook 'Seawulf'
